chat.codes

An Advanced Code Discussion Tool

Create a New Channel

1. Open Google Chrome and go to “chat.codes/“+ChannelName

2. Input Username and push the button “Go”

e.g. http://chat.codes/firstChannel
Note: Only works with Google Chrome

Join an Existing Channel

•  Same (same instructions as “Create a New Channel” above)

Create an EditorState

•  Click "+" on the upper left part of the window

Note: Only the user who created the channel can create editors

Editor Sync

•  Every user in a given channel sees the same set of editors and cursors

Note: The color of cursor selection is the same as the username color

Send a Message

1. Enter a message into the chat input

2. Press ENTER using keyboard to send a message

Note: As someone is typing their username will shake
Note: If the user has typed something, their name will be followed by “…”

Messages

•  Click a message to see the content of the editor when that message was sent

Note: Earlier versions of code are not editable
Note: Click “show current code” to go back to the current version of code

Edits

When anyone edits code, an “edit” message appears in the chat window

•  Click “(more)” to show the difference of two versions of code

Note: Earlier versions of code are not editable
Note: Click “show current code” to go back to the current version of code

Linking to Code

CodeLink is a link inside a message that refers to certain part of code of certain file

In ChatInput Field, it shows in the form of

[ {text} ]( {FileName}:L{StartRow},{StartColumn}-L{EndRow},{EndColumn} )

When sent, a CodeLink looks just like a link.

•  Creating code links: Select any part of code in the Editor, and a CodeLink for it will appear automatically in ChatInput Field

•  Move your cursor over a CodeLink, the LinkedFile will go back to earlier version and the LinkedCode will be highlighted

•  Click on a CodeLink, and the earlier version of LinkedFile will be shown and LinkedCode will be highlighted for 3 seconds

Note: Press ShowCurrentCode button to show current code

chat.codes

An Advanced Code Discussion Tool

Create a New Channel

1. Open Google Chrome and go to “chat.codes/“+ChannelName

2. Input Username and push the button “Go”

e.g. http://chat.codes/firstChannel
Note: Only works with Google Chrome

Join an Existing Channel

•  Same (same instructions as “Create a New Channel” above)

Create an EditorState

•  Click "+" on the upper left part of the window

Note: Only the user who created the channel can create editors

Editor Sync

•  Every user in a given channel sees the same set of editors and cursors

Note: The color of cursor selection is the same as the username color

Send a Message

1. Enter a message into the chat input

2. Press ENTER using keyboard to send a message

Note: As someone is typing their username will shake
Note: If the user has typed something, their name will be followed by “…”
